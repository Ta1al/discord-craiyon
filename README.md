# DisGPT
Discord Integration for Craiyon (Image Generation AI).

### For self-hosting
##### Clone the repository
```bash
git clone https://github.com/Ta1al/disgpt
```
##### Install dependencies
```bash
npm install
```
##### Build
```bash
npm run build
```
##### Rename `example.env` to `.env` and fill in the values
##### Run the bot
```bash
npm start
```
